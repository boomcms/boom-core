Readability Measure

The page readability measure gives the SMOG grade for the page. This gives an indication of the number of years education needed to read the page.

Some caveats to consider:

  • The readability score is generated from the standfirst and bodycopy text. If this text isn't present for a particular template, or is very short, then the readability score will not be useful.
  • A readability score will only be accurate for text which is in English.
